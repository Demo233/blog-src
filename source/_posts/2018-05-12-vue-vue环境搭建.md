## 目录

- 安装vue脚手架
- 初始化项目
- 运行项目

## 安装vue脚手架

> sudo npm install -g vue-cli

#初始化项目

使用list可以查看已存在的模板。

> vue list

用init命令执行默认模板是webpack来初始化项目sell

> vue init webpack sell

## 运行项目

使用npm 构建项目(在init时指定的是yarn，这里就使用yarn install)，install会下载项目具备的依赖包

> npm install

然后就可以运行项目了

> npm run dev

访问localhost:8080
