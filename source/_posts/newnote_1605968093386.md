### 一. 邻接矩阵无向图的介绍

邻接矩阵无向图是一种数据结构，其利用数组存储顶点，二维数组存储顶点之间的关系来存储数据。如下面所示

### 二. 邻接矩阵无向图的代码说明

#### 1. 基本定义

```java
public class Graph<T> {
        // 顶点
        private ArrayList<T> vertexs;
        // 边
        private int[][] edge;
	...
}
```

#### 2. 
vertexs 用于存储顶点值，edge用于存储各个顶点的关系，我们把这种关系称之为``边``
